== Description ==
It is a plugin which allows you to manage your online appointment, reservation and rental bookings.  

[FREE VERSION DEMO LINK](http://rentalbookingfree.phoeniixxdemo.com/shop/) | [FREE VERSION DOCUMENTATION LINK](http://rentalbookingfree.phoeniixxdemo.com/wp-content/uploads/2018/04/Appointment-Plugin-Free-Docs.pdf)

= Features =

* The plugin is compatible with our <strong>Support Ticket System By Phoeniixx plugin</strong>.
* **Day** and **Night** calculation modes.
* Option to select Dates.
* Product based booking settings 
* Option to show to from and to calender fields or single calender field.

== Installation ==

1. Install [WordPress](https://wordpress.org/)
2. Go to your WordPress admin panel, and then to Plugins. Click on "Add New" and then upload the zip file of the plugin using the "Upload Plugin" button you can find on top of the screen.
