== Description ==
[IdoSell Booking](http://www.idosell.com/uk/booking/) is an online booking plugin for your WordPress website. It’s an all-in-one reservation management system with a [web-based admin panel](http://www.idosell.com/uk/booking/functions/reservations/) to update your availability & rates, [synchronize with multiple sales channels](http://www.idosell.com/uk/booking/functions/channel-manager/), handle [online payments](http://www.idosell.com/uk/booking/integrations/payments/), send automatic booking confirmations and much more.

= Who can use IdoSell Booking? =
The system is an excellent choice for hotels, vacation rentals, B&Bs, apartments, flats, camping sites, tour agencies, yacht charters. With IdoSell Booking plugin, your clients can make daytime, overnight, weekly or even monthly reservations.You can sell single items, multi-item sets and special packages. Your inventory updates automatically after each new reservation is made.

== Installation ==

Install the IdoSell Booking plugin in your WordPress dashboard. Go to '*Plugins/Add New*' and search for '*IdoSell Booking*'. You can also install it manually by downloading the plugin and unzipping it into your /wp-content/plugins/ directory. The plugin needs to be activated via the '*Plugins*' tab in WordPress.

= Configuration: =
- Create your IdoSell Booking account.
- Set up the account by adding inventory and rates in the admin panel → [Configuration Guide](http://www.idosell.com/uk/booking/guide/quick-start-guide/).
- Embed IdoSell Booking Widget on your WordPress by adding the shortcode "[idosellbooking|widget/]" (see [our instruction](http://www.idosell.com/uk/booking/integrations/webpages/wordpress/) to learn more).
- More customized options and ready-to-use codes are available in your IdoSell Booking panel ([contact us](http://www.idosell.com/uk/booking/contact/) in case of any questions).
